  如何添加新 Drama

  编辑 data/dramas.json，添加新条目：
  {
    "id": "unique-id",
    "title": "作品名",
    "circle": "社团名",
    "cv": ["CV1", "CV2"],
    "cover": "covers/xxx.webp",
    "password": null,  // 或 "your-password"
    "tracks": [...]
  }

  启动方式

  cd DramaPlayer
  npx serve -p 8080    # 推荐：支持 Range 请求，进度恢复正常工作
  # 或 python -m http.server 8080  # 注意：不支持 seek，进度恢复不工作
  # 访问 http://localhost:8080

你可以调用你的frontend-design和feature-dev plug-ins

  - 卡片样式（加边框装饰、悬浮效果等）
  - Logo 设计
  - 添加装饰元素

播放进度系统 (已实现)
  - 详情页显示每轨播放状态：「続き X:XX」或「再生済み」
  - 点击轨道恢复上次播放位置
  - 播放按钮智能显示：「続きから」/「最初から」/「再生する」
  - 播放完一轨后标记为「再生済み」，下次从头播放
  - 最后一轨播放完返回详情页
  - 进度只会向前保存，不会被更小的值覆盖
